# Semester 6

Semester 6 brought together application development, business intelligence, high-performance processing, and data engineering. Across these courses, I moved from learning individual tools to understanding how complete digital and data solutions are planned, built, analyzed, and improved.

## Courses

| Course | What I Gained |
| --- | --- |
| [Application Development](Application%20Developement/) | Improved my understanding of application planning, documentation, user needs, and system modules |
| [Business Intelligence](Business%20Intelligence/) | Learned to prepare data, build dashboards, create workflows, and communicate insights for decision-making |
| [High Performance Data Processing](High%20Performance%20Data%20Processing/) | Developed stronger awareness of performance, scalability, workload handling, and technical reporting |
| [Special Topic in Data Engineering](Special%20Topic%20in%20Data%20Engineering/) | Practiced cloud tools, Spark concepts, pipeline thinking, notebooks, dashboards, and industry-style data engineering |

## Semester Reflection

This semester helped me see how different computing areas connect. Application Development strengthened my planning and user-focused thinking. Business Intelligence taught me how to transform data into meaningful insights. High Performance Data Processing improved my understanding of efficient and scalable systems. Special Topic in Data Engineering gave me practical experience with modern data workflows, cloud tools, and pipeline-based projects.

Overall, I gained a more complete view of how software and data systems support real decisions. I became more confident in documenting work, using technical tools, and explaining the value of a solution in a clear and human-friendly way.
