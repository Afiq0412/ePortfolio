# Special Topic in Data Engineering

## Course Overview

Special Topic in Data Engineering exposed me to modern data engineering practices, including cloud tools, data pipelines, AI-assisted workflows, and project-based data solutions. The folder contains tutorials, notebooks, reports, dashboard files, and the PPG industry project materials.

## What I Gained

I gained hands-on experience with the data engineering workflow from ingestion to transformation and reporting. The course helped me understand how tools such as Microsoft Azure, Apache Spark, notebooks, Power BI, and pipeline concepts can be combined to solve real data problems.

## Skills Practiced

| Area | Growth |
| --- | --- |
| Cloud data tools | Exploring Microsoft Azure for data-related tasks |
| Big-data processing | Practicing Apache Spark concepts through tutorials |
| Data pipelines | Understanding bronze, silver, and project workflow stages |
| AI-assisted development | Learning how AI tools can support pipeline design and analysis |
| Industry project work | Applying data engineering ideas to PPG-related datasets and reporting |

## Reflection

This course gave me one of the clearest links between classroom learning and industry-style data work. I learned that data engineering requires both technical skill and structure: knowing where data comes from, how it is transformed, how quality is maintained, and how the final result supports decision-making.
