# ⚙️ Semester 4

> A practical semester where programming, analytics, operating systems, communication, and entrepreneurship started to feel more connected.

## ✨ Semester Focus

Semester 4 strengthened my programming maturity and introduced me to more advanced thinking in software, data, operating systems, communication, and entrepreneurship. Object Oriented Programming helped me understand how Java programs can be organized using classes, objects, inheritance, and reusable design principles. Data Analytic Programming and Data Mining exposed me to the value of data and how programming can be used to find patterns and insights. Operating System helped me understand what happens behind the scenes when computers manage processes, memory, files, and resources. At the same time, Professional Communication Skill, Philosophy and Current Issues, and Technopreneurship helped me build awareness of communication, society, ethics, and business opportunities related to technology.

## 📝 Reflection

Semester 4 helped me grow from basic programming into more mature and organized software thinking. I became more comfortable with Java and object-oriented concepts, which improved the way I structure code and understand reusable program design. I also learned that data can provide meaningful knowledge when it is analyzed properly, and that operating systems play an important role in supporting every program we run. Beyond technical knowledge, this semester also improved my ability to communicate ideas, think about current issues, and understand how technology can become a business opportunity. Overall, it helped me become more balanced as a student by developing both technical skills and broader professional awareness.
