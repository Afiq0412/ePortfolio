# 🧠 Semester 3

> A core computer science semester focused on structure, systems, databases, networking, and software development practice.

## ✨ Semester Focus

Semester 3 moved deeper into the core areas of computer science and gave me a better understanding of how larger systems are created and maintained. Data Structure and Algorithm taught me how data can be organized efficiently and how algorithm choices affect program performance. Database introduced stronger concepts of storing, managing, and retrieving structured data, while Network Communication helped me understand how devices exchange information across networks. Software Engineering and System Development taught me that building software requires planning, documentation, teamwork, requirements analysis, and proper development methods. This semester helped me see how different technical components work together to support reliable and maintainable systems.

## 📝 Reflection

Semester 3 improved my ability to think in a more structured and systematic way. I learned that strong software systems are not created randomly; they depend on good algorithms, suitable data structures, well-designed databases, reliable network understanding, and disciplined software engineering practices. I also gained more experience reading, writing, and organizing technical work through labs, assignments, and project documentation. This semester made me more aware of the importance of efficiency, maintainability, and planning in software development. It helped me grow from simply solving individual programming problems into thinking about how complete systems are designed, connected, and improved over time.
